# StayPune

StayPune is a web application that helps people discover rental opportunities and accommodation discussions related to Pune.
It aggregates recent housing-related posts and presents them in a simple, clean interface.

## 🚀 Features

* Displays latest rental-related posts
* Filters posts relevant to flats, BHK, rent, and roommates
* Clean card-based UI for browsing listings

## 🛠 Tech Stack

Frontend

* React
* Vite
* TypeScript

Data Fetching

* TanStack React Query

Code Quality

* ESLint

Package Manager

* pnpm

## 📦 Project Setup

Clone the repository:

```bash
git clone https://github.com/Charukesh-0316/StayPune.git
cd StayPune/staypune
```

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

The application will run at:

```
http://localhost:5173
```
